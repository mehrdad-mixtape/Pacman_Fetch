## How to use?
- Look at the **bin** and run pacmanfetch (Just Linux!)
    ```bash
    ./bin/pacmanfetch
    ```
- Run pacmanfetch.py
    1. Install requirements
        - Linux or mac:
            ```bash
            pip3 install -r requirements.txt
            ```
        - Windows:
            ```bash
            pip install -r requirements.txt
            ```
    2. Run
        - Linux or mac:
            ```bash
            python3 pacmanfetch.py
            ```
        - Windows:
            ```bash
            python pacmanfetch.py
            ```
